### Project Setup
```bash
git clone git@github.com:HelipengTony/ouorz-next.git
cd ouorz-next
yarn install
yarn run dev
```

<br/>

## Deployment
### Next.js Server-side rendering

Create a `.env` file with your configurations, then run:

```bash
yarn run build
yarn run start
```
